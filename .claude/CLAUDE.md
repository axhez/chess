# Chess Project

Single-file web chess game (`index.html`). All HTML, CSS, and JS lives in that one file — do not split into multiple files.

## Stack
- Vanilla JS, no frameworks or libraries
- Web Audio API for sounds (synthesized, no external audio files)
- CSS animations for board flip and piece movement

## Conventions
- Keep everything in `chess.html`
- Sounds are generated via Web Audio API — no external sound files
- Board supports auto-flip per turn and manual override
- Black pieces use CSS color `#1a1a1a`; white pieces use default Unicode rendering
